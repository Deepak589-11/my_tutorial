HTML Basics

HyperText Markup Language is the standard markup language for documents designed to be displayed in a web browser. It defines the structure and content of a web page.

Boilerplate
The essential tags required to create any HTML document.
<!DOCTYPE html>
<html lang="en">
<head>
    <title>Page Title</title>
</head>
<body>
    <!-- Content goes here -->
</body>
</html>
Headings & Text
Used for titles and paragraphs to format text.
<h1>Main Heading</h1>
<h2>Subheading</h2>
<!-- h1 through h6 available -->

<p>This is a paragraph.</p>
<strong>Bold text</strong>
<em>Italic text</em>
Links & Media
Connecting pages and embedding images.
<!-- Anchor (Link) -->
<a href="https://example.com">Visit Site</a>

<!-- Image -->
<img src="image.jpg" alt="Description">
Containers
Elements used to group other elements together.
<!-- Block-level container -->
<div class="container">
    <p>Grouped content</p>
</div>

<!-- Inline container -->
<span style="color: red;">Highlight</span>
Lists
Used to group related items.
<!-- Unordered List -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<!-- Ordered List -->
<ol>
    <li>First</li>
    <li>Second</li>
</ol>
Forms & Inputs
Collecting user data.
<form action="/submit">
    <label for="name">Name:</label>
    <input type="text" id="name">
    
    <input type="password">
    <input type="checkbox">
    
    <button type="submit">Send</button>
</form>
Tables
Displaying data in rows and columns.
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data A</td>
        <td>Data B</td>
    </tr>
</table>
Semantic Tags
Elements with meaning for better accessibility and SEO.
<header>Top area</header>
<nav>Navigation links</nav>
<main>Main content</main>
<article>Independent post</article>
<section>Thematic grouping</section>
<footer>Bottom area</footer>

CSS Fundamentals

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in HTML. It controls the layout, colors, and fonts.

Selectors
How to target HTML elements to apply styles.
/* Tag selector */
p { color: white; }

/* Class selector */
.my-class { margin: 10px; }

/* ID selector */
#my-id { font-size: 20px; }

/* All elements */
* { box-sizing: border-box; }
Box Model
The core concept of layout: content, padding, borders, and margins.
.box {
    width: 200px;
    height: 100px;
    padding: 20px;   /* Inside space */
    border: 2px solid red;
    margin: 15px;    /* Outside space */
}
Flexbox
A one-dimensional layout method for arranging items in rows or columns.
.container {
    display: flex;
    justify-content: center; /* Horiz align */
    align-items: center;     /* Vert align */
    flex-direction: row;     /* Default */
    gap: 1rem;               /* Spacing */
}
Transitions & Hover
Smooth animations when state changes.
.btn {
    background: blue;
    transition: all 0.3s ease;
}

.btn:hover {
    background: darkblue;
    transform: scale(1.05);
}
CSS Grid
A powerful 2D layout system for rows and columns.
.grid-container {
    display: grid;
    /* 3 equal columns */
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}
.item {
    /* Span 2 columns */
    grid-column: span 2; 
}
Positioning
Control where elements appear on screen.
.element {
    /* Flow normally */
    position: static;
    
    /* Relative to normal position */
    position: relative; top: 10px;
    
    /* Relative to nearest positioned ancestor */
    position: absolute; right: 0;
    
    /* Fixed to viewport */
    position: fixed; bottom: 0;
}
Typography
Styling text and fonts.
p {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    font-weight: bold;
    line-height: 1.5;
    text-align: center;
    text-transform: uppercase;
    text-decoration: underline;
}
Media Queries
Responsive design for different screen sizes.
/* Styles for mobile (default) */
.col { width: 100%; }

/* Tablet and larger */
@media (min-width: 768px) {
    .col { width: 50%; }
}

/* Desktop and larger */
@media (min-width: 1024px) {
    .col { width: 25%; }
}

JavaScript Essentials

JavaScript is a programming language that allows you to implement complex features on web pages, such as interactive content, dynamic updates, and animations.

Variables & Data Types
Storing information in memory.
// let can be reassigned
let count = 10; 

// const cannot be reassigned
const name = "John"; 

// Data Types
let isTrue = true;     // Boolean
let arr = [1, 2, 3];   // Array
let obj = { age: 25 }; // Object
Functions
Reusable blocks of code.
// Traditional Function
function greet(name) {
    return "Hello " + name;
}

// Arrow Function (Modern)
const add = (a, b) => {
    return a + b;
};
// Implicit return: const add = (a,b) => a+b;
DOM Manipulation
Interacting with HTML elements via JS.
// Select an element
const btn = document.querySelector('.my-btn');

// Change content
btn.textContent = "Clicked!";

// Change style
btn.style.backgroundColor = "red";

// Add a class
btn.classList.add('active');
Events
Responding to user interactions.
const button = document.getElementById('btn');

// Add click listener
button.addEventListener('click', () => {
    alert('Button was clicked!');
});

// Other events: 'mouseover', 'keydown', etc.
Arrays & Methods
Working with lists of data.
const fruits = ['Apple', 'Banana'];
fruits.push('Cherry'); // Add to end

// map: creates a new array
const sizes = fruits.map(f => f.length);

// filter: keeps items that match
const long = fruits.filter(f => f.length > 5);

// forEach: loops over items
fruits.forEach(f => console.log(f));
Objects & JSON
Storing key-value pairs and data transfer.
const user = {
    name: "Alice",
    age: 28,
    greet() { console.log("Hi"); }
};

console.log(user.name); // Alice

// JSON (JavaScript Object Notation)
const str = JSON.stringify(user);
const obj = JSON.parse(str);
Promises & Async
Handling asynchronous operations (like API calls).
// Using Promises
fetch('api/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// Using Async/Await
async function getData() {
    try {
        const res = await fetch('api/data');
        const data = await res.json();
    } catch(err) { console.error(err); }
}
Conditionals & Loops
Control flow in logic.
// If-Else
if (score > 90) { grade = 'A'; } 
else { grade = 'B'; }

// Ternary Operator
let msg = isTrue ? "Yes" : "No";

// For Loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

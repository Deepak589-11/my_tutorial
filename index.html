HTML Basics

HyperText Markup Language is the standard markup language for documents designed to be displayed in a web browser.

Boilerplate
Document Foundation
The essential tags required to create any HTML document. It defines the document type and contains the head (metadata) and body (visible content).
Example Code:
<!DOCTYPE html>
<html lang="en">
<head>
    <title>Page Title</title>
</head>
<body>
    <!-- Content here -->
</body>
</html>
Headings & Text
Typography Elements
Used to structure text content logically.
  • <h1> to <h6>: Headings (h1 is most important).
  • <p>: Standard paragraph text.
  • <strong> / <em>: Bold and italic emphasis.
Example Code:
<h1>Main Heading</h1>
<p>This is a paragraph.</p>
<strong>Bold text</strong>
Links & Media
Interactive Content
Elements used to connect pages or embed media.
  • <a>: Anchor tag for creating hyperlinks.
  • <img>: Embeds an image (requires src and alt attributes).
Example Code:
<a href="url.com">Click Here</a>
<img src="image.jpg" alt="Description">
Containers
Layout Grouping
Generic elements used to group other elements, usually for styling purposes.
  • <div>: A block-level container (takes up full width).
  • <span>: An inline container (wraps tightly around content).
Example Code:
<div class="box">
    <span>Highlighted</span> text.
</div>
Lists
Data Grouping
Used to group a set of related items.
  • <ul>: Unordered list (bullet points).
  • <ol>: Ordered list (numbered).
  • <li>: List item (used inside ul/ol).
Example Code:
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>
Forms & Inputs
User Interaction
Used to collect user input.
  • <form>: The wrapper for input elements.
  • <input>: Data field (text, password, checkbox).
  • <button>: Triggers form submission.
Example Code:
<form>
    <input type="text" placeholder="Name">
    <button type="submit">Send</button>
</form>
Tables
Grid Data
Used for displaying tabular data in rows and columns.
  • <table>: The main container.
  • <tr>: Table row.
  • <th> / <td>: Header cell / Standard data cell.
Example Code:
<table>
    <tr>
        <th>Name</th>
        <td>Alice</td>
    </tr>
</table>
Semantic Tags
Meaningful Structure
Tags that describe their meaning to both the browser and the developer (improves SEO & Accessibility).
  • <header> / <footer>: Top/bottom of a page.
  • <nav> / <main>: Navigation / primary content.
Example Code:
<header>Logo & Nav</header>
<main>
    <article>Blog Post</article>
</main>

CSS Fundamentals

Cascading Style Sheets describes the presentation of a document written in HTML.

Selectors
Targeting Elements
How you select which HTML elements to apply styles to.
  • Tag: Targets all elements of a type (e.g., p).
  • Class (.): Targets elements with a specific class.
  • ID (#): Targets a single unique element.
Example Code:
p { color: white; }
.my-class { margin: 10px; }
#my-id { font-size: 20px; }
Box Model
Spacing & Layout
Every element is a rectangular box. The box model defines how spacing is calculated.
  • Padding: Space inside the border.
  • Border: The line around the element.
  • Margin: Space outside the border.
Example Code:
.box {
    padding: 20px;
    border: 2px solid red;
    margin: 15px;
}
Flexbox
1D Layout System
A layout model for aligning items in a single row or column.
  • justify-content: Aligns along the main axis.
  • align-items: Aligns along the cross axis.
Example Code:
.container {
    display: flex;
    justify-content: center;
    align-items: center;
}
Transitions & Hover
Animations
Used to create smooth changes when an element's state changes.
  • transition: Defines duration and easing.
  • :hover Pseudo-class triggered when mouse is over element.
Example Code:
.btn { transition: all 0.3s; }
.btn:hover {
    transform: scale(1.05);
}
CSS Grid
2D Layout System
A powerful layout system for designing complex web pages using rows and columns simultaneously.
Example Code:
.grid {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 20px;
}
Positioning
Element Flow
Controls how an element is placed within the document.
  • Relative: Moved relative to normal position.
  • Absolute: Removed from flow, positioned to nearest parent.
  • Fixed: Stuck to the viewport.
Example Code:
.modal {
    position: fixed;
    top: 50%;
    left: 50%;
}
Typography
Text Styling
Properties to control the appearance of text.
  • font-family: Defines the typeface.
  • line-height: Vertical spacing between lines.
Example Code:
p {
    font-family: 'Arial', sans-serif;
    font-size: 16px;
    text-align: center;
}
Media Queries
Responsive Design
Applies CSS rules only when the screen meets certain size conditions (like mobile vs desktop).
Example Code:
@media (min-width: 768px) {
    .container {
        width: 50%;
    }
}

JavaScript Essentials

JavaScript implements complex features, interactive content, and dynamic updates.

Variables
Data Storage
Containers for storing data values.
  • let: Variable that can be reassigned later.
  • const: Constant variable that cannot be changed.
Example Code:
let score = 10;
score = 15; // Valid

const name = "John";
// name = "Doe"; -> Error
Functions
Reusable Logic
Blocks of code designed to perform a particular task. They run when "called" or "invoked". Modern JS often uses "Arrow" syntax.
Example Code:
// Arrow Function
const add = (a, b) => {
    return a + b;
};
console.log(add(5, 2)); // 7
DOM Manipulation
Changing HTML
Using JS to select and modify HTML elements on the page (Document Object Model).
Example Code:
// Select element
const btn = document.querySelector('.btn');

// Change text
btn.textContent = "Clicked!";
Events
User Actions
Reacting to things the user does, such as clicking, scrolling, or pressing keys.
Example Code:
const btn = document.getElementById('btn');

btn.addEventListener('click', () => {
    alert('Hello!');
});
Arrays & Methods
List Operations
Special variables that hold more than one value.
  • map(): Transforms every item.
  • filter(): Keeps items matching a condition.
Example Code:
const nums = [1, 2, 3];
const doubled = nums.map(n => n * 2);
// doubled is [2, 4, 6]
Objects
Key-Value Data
Data structures containing properties (key-value pairs) and methods (functions).
Example Code:
const user = {
    name: "Alice",
    age: 28,
};
console.log(user.name); // Alice
Promises & Async
Asynchronous Operations
Used for tasks that take time (like fetching data from a server) without freezing the website. async/await is the modern syntax.
Example Code:
async function getData() {
    const res = await fetch('/api');
    const data = await res.json();
    console.log(data);
}
Conditionals
Logic Flow
Executing different code based on whether a condition is true or false.
Example Code:
if (score > 90) { 
    grade = 'A'; 
} else { 
    grade = 'B'; 
}
